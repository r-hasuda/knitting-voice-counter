# 🧶 音声編み物カウンター (Voice Knitting Counter)

声だけで「段数」や「目数」を数えられる、ハンズフリーの編み物カウンターアプリです。
編み物中にスマホやカウンターに触れる必要がなく、作業に集中できます！

🌐 **[このアプリを実際に使ってみる (Demo)](#)**  
*(※GitHub Pages等で公開後、ここの `#` を公開URLに書き換えてください)*

## ✨ 特徴 (Features)
* **🗣️ 音声で完全操作**
  * 「はい」と言うと目数をカウントアップ！
  * 「次」と言うと段数が進みます。
* **🛠️ ３つの編み方モード対応**
  * 「細編み」「増やし目」「減らし目」のモードがあります。
  * 特に「増やし目」モードでは、自動で「1/2回目」「2/2回目」をサポート。
* **↩️ 間違えても安心のアンドゥ機能**
  * 「間違えた」「もどして」と言うと、一つ前の状態に戻ります（最大50回前まで記憶）。
* **💾 データ自動保存**
  * ブラウザを閉じても、前回の続きからすぐに編み始められます。
* **📱 スマホの画面が消えない**
  * 音声待ちの間、スリープ（画面暗転）を防ぐ機能（Wake Lock API）を搭載。

## 🎤 音声コマンド一覧

| やりたいこと | 声のかけ方 |
| :--- | :--- |
| **目数を＋1する** | 「はい」 |
| **次の段へ進む** | 「次 (つぎ)」 |
| **細編みモードにする** | 「細編み (こまあみ)」 |
| **増やし目モードにする** | 「増やし目 (ふやしめ)」 |
| **減らし目モードにする** | 「減らし目 (へらしめ)」 |
| **1つ前に戻す** | 「戻して」「間違えた」など |

*※音声認識がオフの場合は、画面のボタンや文字を直接タップすることでも操作可能です。*
